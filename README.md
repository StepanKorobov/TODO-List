# TODO-List

Консольное приложение для работы с задачами

## Функционал:

1) Вывод всех задач.
2) Добавление задачи.
3) Удаление задачи.

- При запуске приложения происходит чтение из файла "TodoList.txt".
- При добавлении/удалении происходит запись в файл "TodoList.txt".
- Автосохранение изменений.
